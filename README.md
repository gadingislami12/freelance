# Gading Islami | Developer Portfolio

Welcome to my personal developer portfolio! This project serves as a central hub to showcase my skills, projects, and professional background. It is designed with a sleek, modern aesthetic and built to dynamically highlight my ongoing work.

## ✨ Features

- **Dynamic GitHub Integration**: Automatically fetches and displays my most recently updated public repositories using the GitHub API.
- **Modern Premium Design**: Utilizes a dark mode theme with glassmorphism UI elements (frosted glass effects) for a professional, "wow" factor.
- **Interactive UI**: Includes smooth gradients, subtle micro-animations on hover, and an engaging layout.
- **Live Previews**: Automatically links to deployed live versions of my projects (e.g., GitHub Pages or custom domains like InfinityFree).
- **Responsive**: Fully responsive design that looks great on mobile, tablet, and desktop devices.
- **Optimized**: Fast loading, no heavy frameworks required. Built with vanilla HTML, CSS, and JavaScript.

## 🚀 Tech Stack

- **HTML5**: Semantic and clean structure.
- **CSS3**: Custom variables, flexbox, CSS grid, and backdrop-filter for glassmorphism.
- **JavaScript (ES6+)**: Asynchronous fetching (`fetch API`) to dynamically load GitHub data.
